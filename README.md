# Engineering Notes

## 概要
本リポジトリは、エンジニアリングに関する学習内容を体系的に整理・蓄積することを目的としたナレッジベースである。  
Web開発・アプリ開発・アルゴリズム・設計思想などを横断的に扱い、理解の深化と実務レベルへの到達を目指す。

---

## 背景
エンジニアリングの学習においては、知識が断片化しやすく、体系的な理解が難しいという課題がある。  
また、学習内容をアウトプットせずにインプットだけを繰り返すと、定着しないという問題もある。

これらを解決するために、

- 学習内容を構造化して記録する
- GitHubでバージョン管理する
- 実務に近い開発フロー（PR・CI）で運用する

という方針で本リポジトリを構築している。

---

## 目的

本リポジトリの目的は以下の通りである。

- 知識の体系化（断片的な理解の解消）
- アウトプットによる理解の定着
- 実務レベルの開発フローの習得
- ポートフォリオとしての活用
